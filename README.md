# Personal Portfolio Website

## Overview

Professional portfolio website built using Angular, TypeScript, HTML, CSS, and Bootstrap.

The website showcases:

- Professional Experience
- Technical Skills
- Projects
- Resume
- Contact Information

## Technologies

- Angular
- TypeScript
- HTML5
- CSS3
- Bootstrap

## Features

- Responsive Design
- Project Showcase
- Resume Download
- Contact Information
- Modern User Interface

## Installation

```bash
npm install
ng serve
```

Visit:

```text
http://localhost:4200
```
